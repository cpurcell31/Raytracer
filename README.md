### Ray Tracer

#### Features
* Pinhole Camera System
* Point Lights
* Diffuse Shading
* Jittered multisampling
* Out-of-gamut handling
* Sphere, Box, Plane geometries
* Shadows 
* Area Lights 
* Mirror Reflections 
* Ambient Occlusion 
* Glossy Reflections 
* Meshes 
* Multithreading
